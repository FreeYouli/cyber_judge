## 系统架构

```
📂 数据层 → 🛠️ 工具层 → 🧠 推理层 → 🎨 展示层
(149案例)  (4个工具)   (DeepSeek)  (PyQt5)
```

- 用户输入 → 情感分析 → 关键词提取
- → 相似案例检索 → LLM综合推理
- → 结构化判决书 + 雷达图

---

## 四大工具

| 工具 | 功能 | 技术 |
|------|------|------|
| 情感分析 | 判断双方情绪倾向 | DeepSeek API |
| 关键词提取 | 抽取争议焦点 | jieba TF-IDF |
| 相似案例检索 | 检索历史Top-3 | TF-IDF + 余弦相似度 |
| LLM综合推理 | 生成判决书 | DeepSeek-chat |
